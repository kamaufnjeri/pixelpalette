==========

Terminal
--------
to enter virtual enviromnent
"source my_env_name/bin/activate or my_env_name/Scripts/activate"

"git checkout my_branch_name"
work on the project

after everything - run migrate in case of changes to the database - "flask db migrate -m 'commit message'"
								  - "flask db upgrade to activate the changes"
								  - "incase of errors flask db downgrade to revert the changes"
		 - update the requirements.txt file with the new python packages/modules installed
		 	"pip freeze > requirements.txt"
- deactivate to leave virtual environment
